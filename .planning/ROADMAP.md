### Phase Details

**Phase 1: Ingestor Pipeline**
Goal: Functional Python ingestor pipeline with Supabase sync
Requirements: ING-01, ING-02, ING-03, ING-04
Success criteria:
1. Python environment and scripts established.
2. Data fetched, filtered, and transformed correctly.
3. Data successfully upserted to Supabase.

**Phase 2: Visual Foundation**
Goal: Basic Resium globe with data mapping
Requirements: VIS-01, VIS-02
Success criteria:
1. Next.js app set up with Resium.
2. Globe renders with Cesium tokens.
3. Points rendered from Supabase data.

**Phase 3: Impact Visualization**
Goal: Visualizing felt radius and impact zones
Requirements: VIS-03
Success criteria:
1. Ellipses/Circles rendered for impact zones.
2. Pulsing animation effect.

**Phase 4: UI & Polish**
Goal: Metadata panel and final polish
Requirements: UI-01
Success criteria:
1. Side panel displays details on selection.
2. UI is responsive and clean.
